# Blue Hunter

Projeto desenvolvido com base no desafio [Blue Hunter](https://github.com/mpguedes/blue-hunter).

## Tecnologia

 * Angular 4
 * Boostrap 3
 * Karma & Jasmine
 * Less
 * NodeJs
 * Typescript
 * Webpack 2

## Scripts

Instalar dependências:
```bash
$ npm install
```
Start aplicação
```bash
$ npm start
```
Testes
```bash
$ npm run test
```
Build projeto
```bash
$ npm run build
```
Start do projeto a partir do build
```bash
$ npm run serve
```
